# AffirmationVibration

## Short Description
A simple, calming application that generates affirmations designed to help users raise their vibration based on the Map of Consciousness.

## Features
- About section explaining the purpose of the app
- Affirmation generator based on selected mood
- Smooth transitions and dynamic background colors
- Responsive layout for desktop and mobile

## Tech Used
- React
- JavaScript
- CSS
- Node + Express (small REST-inspired API)
